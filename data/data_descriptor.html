Category: Factuality (audit/factuality)

This category assesses the accuracy of the retrieved data, determining whether the authors and auxiliary data returned by the LLM exist and are correctly attributed to the specified scholar.

File: factuality_author.csv (audit/factuality)

Basic factuality analysis with run metadata and author verification flags. Contains 32,445 rows across all recommendation tasks.

Field Name Description
run_idUnique identifier linking related analysis runs across files (format: {model}_{task}_{date}_{time}).
dateDate the task was performed.
timeTime the task was performed.
modelThe model used for the task.
task_nameThe name of the task performed (e.g., epoch, field, seniority, top_k, twins).
task_paramParameters used in the task (e.g., 1950s, PER, early_career, top_5).
task_attemptThe attempt number (1-3) for each request, as models were allowed up to 3 attempts to generate valid JSON responses.
result_valid_flagIndicates whether the result is valid.
nameThe name of the recommended scholar.
yearsYears of activity as reported by the LLM in their response (only populated for epoch tasks, otherwise empty).
doiDigital Object Identifier as provided by the LLM in their response (only populated for field tasks, otherwise empty).
career_ageCareer age as estimated by the LLM in their response (only populated for seniority tasks, otherwise empty).
clean_nameCleaned name of the author for standardization.
id_author_oaIdentifier for the author's profile in OpenAlex (only if the author was found among all APS publications).
is_in_apsBoolean indicating whether the author was found in the APS database.

Category: Similarity (audit/similarity)

This category measures the similarity across various metrics, focusing on how similar the recommended authors are to each other and the requestor. All similarity files share the same structure with consistent metrics across tasks.

Files: similarity_epoch.csv, similarity_field.csv, similarity_seniority.csv, similarity_top_k.csv, similarity_twins.csv (audit/similarity)

These files contain consistent similarity metrics across all task types, enabling comparative analysis of recommendation patterns.

Field Name Description
run_idUnique identifier linking related analysis runs across files.
modelIdentifier of the recommendation model used.
task_nameName of the evaluation task.
task_paramParameters specific to the evaluation task.
dateDate when the task was executed.
timeTime when the task was executed.
n_name_recommendationsTotal number of names recommended.
n_unique_names_recommendationsNumber of unique names in the recommendations.
n_unique_author_recommendationsNumber of distinct authors recommended.
n_author_hallucinationsNumber of recommended authors not found in reference data.
name_similaritySimilarity score between recommended and target names.
gender_diversityDiversity index based on gender of recommended authors.
ethnicity_diversityDiversity index based on ethnicity of recommended authors.
aps_scholarly_similarityScholarly similarity score using APS data.
oa_scholarly_similarityScholarly similarity score using OpenAlex data.
scholarly_similarityCombined scholarly similarity score.
aps_career_age_similarityCareer age similarity score using APS data.
oa_career_age_similarityCareer age similarity score using OpenAlex data.
career_age_similarityCombined career age similarity score.
institutions_shareProportion of institutional overlap between recommended and reference authors.
country_of_affiliation_shareProportion of country overlap between recommended and reference authors.
coauthors_shareProportion of shared coauthors with target authors.
coauthors_recommended_shareShare of recommended authors who have coauthored with each other.
recommended_author_pairs_are_coauthorsBoolean indicator whether all recommended pairs are coauthors.
